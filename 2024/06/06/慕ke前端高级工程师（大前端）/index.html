一、HTML (超文本标记语言)

HTML 是构建Web页面的基础语言，用于定义页面的结构和内容。

1.1 基本概念

  • 标签：HTML 由一系列标签组成，每个标签描述一种内容或页面元素。
  • 属性：标签可以包含属性，用于提供额外的信息或配置。

1.2 常用标签

  • **<div>**：定义一个块级元素。
  • **<span>**：定义一个内联元素。
  • **<a>**：定义一个超链接。
  • **<img>**：定义一个图像。
  • **<form>**：定义一个表单。
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>HTML Example</title>
</head>
<body>
    <div>
        <h1>Welcome to My Website</h1>
        <p>This is a paragraph.</p>
        <a href="https://example.com">Visit Example</a>
    </div>
</body>
</html>

二、CSS (层叠样式表)

CSS 用于控制HTML元素的外观和布局，使页面更加美观和用户友好。

2.1 基本概念

  • 选择器：用于选择HTML元素。
  • 属性和值：定义元素的样式属性及其值。

2.2 常用属性

  • 颜色和背景：color, background-color
  • 布局：margin, padding, display, position
  • 字体：font-family, font-size, font-weight
body {
    font-family: Arial, sans-serif;
    background-color: #f0f0f0;
    color: #333;
}

h1 {
    color: #007bff;
}

a {
    color: #007bff;
    text-decoration: none;
}

三、JavaScript

JavaScript 是一种高级编程语言，用于实现网页的动态功能。

3.1 基本概念

  • 变量和数据类型：let, const, number, string, boolean
  • 函数：定义和调用函数。
  • 事件：响应用户操作，如点击、提交等。

3.2 DOM 操作

  • 选择元素：document.getElementById(), document.querySelector()
  • 修改内容：element.innerHTML, element.textContent
  • 事件监听：element.addEventListener()
document.addEventListener('DOMContentLoaded', () => {
    const button = document.querySelector('button');
    button.addEventListener('click', () => {
        alert('Button was clicked!');
    });
});

四、前端框架和库

4.1 React

React 是一个用于构建用户界面的JavaScript库，特别适用于构建单页应用。

import React from 'react';
import ReactDOM from 'react-dom';

function App() {
    return (
        <div>
            <h1>Hello, React!</h1>
            <p>This is a simple React component.</p>
        </div>
    );
}

ReactDOM.render(<App />, document.getElementById('root'));

4.2 Vue.js

Vue.js 是一个渐进式JavaScript框架，用于构建用户界面。

<script src="https://cdn.jsdelivr.net/npm/vue@2"></script>

<div id="app">
  <h1>{{ message }}</h1>
</div>

<script>
new Vue({
  el: '#app',
  data: {
    message: 'Hello, Vue!'
  }
})
</script>

4.3 Angular

Angular 是一个平台，用于构建动态Web应用。

import { Component } from '@angular/core';

@Component({
  selector: 'app-root',
  template: `<h1>Hello, Angular!</h1>`,
  styles: [`h1 { color: #007bff; }`]
})
export class AppComponent { }

五、前端构建工具

5.1 Webpack

Webpack 是一个模块打包工具，用于打包JavaScript模块。

// webpack.config.js
module.exports = {
    entry: './src/index.js',
    output: {
        filename: 'bundle.js',
        path: __dirname + '/dist'
    },
    module: {
        rules: [
            {
                test: /\.css$/,
                use: ['style-loader', 'css-loader']
            }
        ]
    }
};

5.2 Babel

Babel 是一个JavaScript编译器，用于将ES6+代码转换为向后兼容的JavaScript版本。

{
  "presets": ["@babel/preset-env"]
}

5.3 NPM

NPM 是JavaScript包管理器，用于管理项目的依赖包。

# 安装依赖包
npm install axios

六、版本控制系统

6.1 Git

Git 是一个分布式版本控制系统，用于管理项目的源代码。

# 初始化Git仓库
git init

# 添加文件到暂存区
git add .

# 提交文件到本地仓库
git commit -m "Initial commit"

# 推送到远程仓库
git remote add origin https://github.com/user/repo.git
git push -u origin master
