2024，Python爬虫系统入门与多领域实战

本文最后更新于 2024年6月12日 下午

在网络爬虫和数据抓取领域，Python的requests库和XPath语法是非常重要的工具。requests库用于发送HTTP请求，从网页上获取数据；XPath则是一种查询语言，用于在XML或HTML文档中查找信息。本教程将详细介绍这两个工具的使用方法及其在实际开发中的应用。

第一部分：Requests库

1.1 Requests库简介

requests是Python的一个HTTP库，允许你发送HTTP请求，使用简单、易于理解的API接口。与Python内置的urllib相比，requests更加直观且功能丰富。

1.2 安装Requests库

你可以使用pip安装requests库：

pip install requests

1.3 发送基本的HTTP请求

1.3.1 GET请求

GET请求用于从服务器获取数据。以下是一个简单的GET请求示例：

import requests

response = requests.get('https://api.github.com')
print(response.status_code)  # 打印响应状态码
print(response.text)  # 打印响应内容

1.3.2 POST请求

POST请求用于向服务器提交数据。以下是一个简单的POST请求示例：

import requests

data = {'key': 'value'}
response = requests.post('https://httpbin.org/post', data=data)
print(response.status_code)
print(response.json())  # 将响应内容解析为JSON格式

1.4 请求头和参数

在发送请求时，你可能需要设置请求头或添加查询参数。

1.4.1 设置请求头

headers = {'User-Agent': 'my-app/0.0.1'}
response = requests.get('https://api.github.com', headers=headers)
print(response.status_code)

1.4.2 添加查询参数

params = {'q': 'requests+language:python'}
response = requests.get('https://api.github.com/search/repositories', params=params)
print(response.json())

1.5 处理响应

requests库提供了多种方法来处理HTTP响应。

1.5.1 响应状态码

response = requests.get('https://api.github.com')
print(response.status_code)  # 200表示成功

1.5.2 响应内容

response = requests.get('https://api.github.com')

# 以文本形式获取响应内容
print(response.text)

# 以JSON格式获取响应内容
print(response.json())

1.5.3 响应头

response = requests.get('https://api.github.com')
print(response.headers)

1.6 超时和错误处理

1.6.1 设置超时

你可以使用timeout参数设置请求的超时时间：

try:
    response = requests.get('https://api.github.com', timeout=5)
    print(response.status_code)
except requests.Timeout:
    print('请求超时')

1.6.2 错误处理

使用requests库处理HTTP错误非常简单：

try:
    response = requests.get('https://api.github.com/notfound')
    response.raise_for_status()
except requests.HTTPError as err:
    print(f'HTTP error occurred: {err}')
except Exception as err:
    print(f'Other error occurred: {err}')

第二部分：XPath语法

2.1 XPath简介

XPath（XML Path Language）是一种用于在XML文档中查找信息的语言。XPath不仅可以在XML中使用，还可以在HTML中应用，非常适合用于网络爬虫数据提取。

2.2 安装XPath工具

为了在Python中使用XPath，你需要安装lxml库：

pip install lxml

2.3 XPath基础语法

2.3.1 基本表达式

XPath表达式用来在XML或HTML文档中选择节点。以下是一些基本的XPath表达式：

/       # 从根节点选择
//      # 从匹配选择的当前节点选择文档中的节点，而不考虑它们的位置
.       # 选择当前节点
..      # 选择当前节点的父节点
@       # 选择属性

2.3.2 具体示例

假设有以下HTML文档：

<html>
  <body>
    <div>
      <h1>Title</h1>
      <p class="content">Paragraph</p>
      <a href="https://example.com">Link</a>
    </div>
  </body>
</html>

以下是一些XPath表达式及其结果：

/html/body/div/h1   # 选择<h1>节点
//p[@class='content']  # 选择class属性为content的<p>节点
//a/@href  # 选择<a>节点的href属性

2.4 在Python中使用XPath

你可以使用lxml库的etree模块来解析HTML文档并使用XPath查询。

2.4.1 解析HTML文档

from lxml import etree

html = '''
<html>
  <body>
    <div>
      <h1>Title</h1>
      <p class="content">Paragraph</p>
      <a href="https://example.com">Link</a>
    </div>
  </body>
</html>
'''

tree = etree.HTML(html)

2.4.2 使用XPath查询

# 选择<h1>节点的文本
h1_text = tree.xpath('/html/body/div/h1/text()')
print(h1_text)  # ['Title']

# 选择class属性为content的<p>节点的文本
p_text = tree.xpath('//p[@class="content"]/text()')
print(p_text)  # ['Paragraph']

# 选择<a>节点的href属性
a_href = tree.xpath('//a/@href')
print(a_href)  # ['https://example.com']

2.5 复杂查询示例

2.5.1 示例HTML文档

假设有以下更复杂的HTML文档：

<html>
  <body>
    <div>
      <h1>Title 1</h1>
      <p class="content">Paragraph 1</p>
    </div>
    <div>
      <h1>Title 2</h1>
      <p class="content">Paragraph 2</p>
      <a href="https://example.com">Link 1</a>
    </div>
    <div>
      <h1>Title 3</h1>
      <p class="content">Paragraph 3</p>
      <a href="https://example.org">Link 2</a>
    </div>
  </body>
</html>

2.5.2 查询所有标题和段落

html = '''
<html>
  <body>
    <div>
      <h1>Title 1</h1>
      <p class="content">Paragraph 1</p>
    </div>
    <div>
      <h1>Title 2</h1>
      <p class="content">Paragraph 2</p>
      <a href="https://example.com">Link 1</a>
    </div>
    <div>
      <h1>Title 3</h1>
      <p class="content">Paragraph 3</p>
      <a href="https://example.org">Link 2</a>
    </div>
  </body>
</html>
'''

tree = etree.HTML(html)

# 查询所有<h1>和<p>的文本
titles = tree.xpath('//h1/text()')
paragraphs = tree.xpath('//p[@class="content"]/text()')

print(titles)  # ['Title 1', 'Title 2', 'Title 3']
print(paragraphs)  # ['Paragraph 1', 'Paragraph 2', 'Paragraph 3']

2.5.3 查询所有链接的URL和文本

# 查询所有<a>节点的href属性和文本
links = tree.xpath('//a')
link_info = [(link.xpath('@href')[0], link.xpath('text()')[0]) for link in links]

print(link_info)  # [('https://example.com', 'Link 1'), ('https://example.org', 'Link 2')]

2.6 结合Requests和XPath进行网络爬虫

2.6.1 示例：爬取一个网页的标题和段落

import requests
from lxml import etree

url = 'https://example.com'

# 发送GET请求
response = requests.get(url)

# 解析HTML文档
tree = etree.HTML(response.text)

# 查询所有<h1>和<p>的文本
titles = tree.xpath('//h1/text()')
paragraphs = tree.xpath('//p[@class="content"]/text()')

print(titles)
print(paragraphs)

2.6.2 示例：爬取一个网页的所有链接

# 查询所有<a>节点的href属性和文本
links = tree.xpath('//a')
link_info = [(link.xpath('@href')[0], link.xpath('text()')[0]) for link in links]

print(link_info)
