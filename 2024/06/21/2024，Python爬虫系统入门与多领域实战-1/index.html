2024，Python爬虫系统入门与多领域实战

本文最后更新于 2024年6月21日 晚上

在现代网络数据采集领域，Selenium和Scrapy是两种常用的工具。Selenium主要用于浏览器自动化测试和处理动态页面，而Scrapy则是一个强大的爬虫框架，适用于大规模的数据抓取。本文将介绍如何使用Selenium进行Web自动化，并展示如何开发Scrapy项目爬虫，最后还将探讨两者结合使用的实战案例。

Selenium实战

Selenium简介

Selenium是一个用于Web应用程序自动化测试的工具，它提供了一系列API来模拟用户操作，如点击、输入、导航等。Selenium支持多种浏览器，如Chrome、Firefox、Safari等，可以用于测试和数据抓取。

安装Selenium

  1. 安装Selenium库：

    pip install selenium

  2. 下载浏览器驱动程序，以Chrome为例：

    • 下载ChromeDriver：https://sites.google.com/a/chromium.org/chromedriver/downloads
    • 将ChromeDriver添加到系统PATH中。

使用Selenium进行Web自动化

以下是一个使用Selenium打开浏览器、访问页面并进行简单操作的示例：

from selenium import webdriver
from selenium.webdriver.common.by import By
from selenium.webdriver.common.keys import Keys

# 创建浏览器对象
driver = webdriver.Chrome()

# 打开页面
driver.get("https://www.example.com")

# 查找元素并进行操作
search_box = driver.find_element(By.NAME, "q")
search_box.send_keys("Selenium")
search_box.send_keys(Keys.RETURN)

# 等待结果加载
driver.implicitly_wait(10)

# 获取结果
results = driver.find_elements(By.CSS_SELECTOR, "h3")
for result in results:
    print(result.text)

# 关闭浏览器
driver.quit()

Selenium高级用法

处理动态页面

很多现代网页是动态加载内容的，Selenium可以很好地处理这些情况。

from selenium.webdriver.support.ui import WebDriverWait
from selenium.webdriver.support import expected_conditions as EC

# 等待某个元素加载完成
element = WebDriverWait(driver, 10).until(
    EC.presence_of_element_located((By.ID, "myDynamicElement"))
)

截图

Selenium支持网页截图功能，可以用于记录测试结果或抓取页面截图。

driver.save_screenshot("screenshot.png")

处理弹窗

alert = driver.switch_to.alert
alert.accept()

开发Scrapy项目爬虫

Scrapy简介

Scrapy是一个用于提取网站数据的应用框架，提供了强大的数据抓取、处理和存储功能。Scrapy支持并发请求，可以高效地抓取大量网页数据。

安装Scrapy

  1. 安装Scrapy：

    pip install scrapy

创建Scrapy项目

创建一个新的Scrapy项目：

scrapy startproject myproject

Scrapy爬虫开发

  1. 生成一个新的爬虫：

    cd myproject
scrapy genspider example example.com

  2. 编辑生成的爬虫文件 example.py：

    import scrapy

class ExampleSpider(scrapy.Spider):
    name = "example"
    allowed_domains = ["example.com"]
    start_urls = [
        'http://www.example.com/',
    ]

    def parse(self, response):
        for title in response.css('h3::text').getall():
            yield {'title': title}

  3. 运行爬虫：

    scrapy crawl example

Scrapy数据存储

Scrapy支持多种数据存储方式，如JSON、CSV、数据库等。

  1. 将数据存储为JSON：

    scrapy crawl example -o output.json

  2. 将数据存储到数据库：

    编辑 pipelines.py 文件：

    import sqlite3

class MyprojectPipeline:

    def open_spider(self, spider):
        self.conn = sqlite3.connect('example.db')
        self.cursor = self.conn.cursor()
        self.cursor.execute('''
            CREATE TABLE IF NOT EXISTS titles (
                id INTEGER PRIMARY KEY AUTOINCREMENT,
                title TEXT
            )
        ''')

    def close_spider(self, spider):
        self.conn.commit()
        self.conn.close()

    def process_item(self, item, spider):
        self.cursor.execute('''
            INSERT INTO titles (title) VALUES (?)
        ''', (item['title'],))
        return item

Selenium与Scrapy结合

在某些情况下，我们可能需要处理动态加载的数据，这时可以结合使用Selenium和Scrapy。

  1. 在Scrapy爬虫中使用Selenium：

    from scrapy import Spider
from scrapy.http import HtmlResponse
from selenium import webdriver

class SeleniumSpider(Spider):
    name = "selenium_spider"
    allowed_domains = ["example.com"]
    start_urls = [
        'http://www.example.com/',
    ]

    def __init__(self):
        self.driver = webdriver.Chrome()

    def parse(self, response):
        self.driver.get(response.url)
        body = self.driver.page_source
        response = HtmlResponse(url=self.driver.current_url, body=body, encoding='utf-8')
        for title in response.css('h3::text').getall():
            yield {'title': title}

    def closed(self, reason):
        self.driver.quit()
